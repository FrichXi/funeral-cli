# Open Source Scope

This repository is the public, runnable distribution of `funeral-cli`.

## Public Keep

- `funeralai/`: application code, prompts, TUI, assets
- `docs/assets/`: public README assets such as the repository logo
- `tests/`: automated tests for runnable behavior
- `README.md`: install and usage documentation
- `LICENSE`: MIT license
- `pyproject.toml`: packaging metadata

## Private Remove

- `CLAUDE.md`: internal AI collaboration and working notes
- `plans/`: internal planning documents
- `articles/`: private article corpus
- `data/`: research artifacts, eval data, graph exports, framework iteration files

## Local Private Archive

For local safety during open-source prep, removed private materials are kept in:

- `.private/repo-private/`

That directory is git-ignored and is not part of the public repository.
